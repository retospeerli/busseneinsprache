```md
# Textgenerator Umtriebsentschädigung

Statische Web-App (GitHub Pages), die aus Personalien und Falldaten einen editierbaren Brieftext zur Bestreitung einer Umtriebsentschädigung / Parkforderung generiert.

## Features
- Vorname + Nachname getrennt
- Tonalität per Emoji-Buttons: 😊 / 😐 / 😑
- Live-Zusammenfassung (Kennzeichen, Datum/Uhrzeit, Ort)
- Copy-to-Clipboard inkl. Feedback
- „Per Outlook versenden“ via mailto (Standard-Mailclient)
- Diktieren (Web Speech API, de-CH) – Chrome/Edge empfohlen

## Struktur
index.html
style.css
app.js
README.md

## Hinweise
- Diktieren funktioniert i.d.R. nur auf HTTPS oder localhost.
- Der Mail-Button öffnet den Standard-Mailclient.
- Die App generiert eine Vorlage; rechtliche Feinjustierungen bitte je nach Einzelfall prüfen.
```
